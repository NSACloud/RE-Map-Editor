This an unfinished Blender addon for editing RE Engine SCN/RSZ files.

Due to people constantly asking me for this, I am releasing it as is.

Only Monster Hunter Rise, RE2 RT and RE4 are supported.

Many buttons and options are not implemented and will not work correctly.

**No support will be provided. If you run into issues, you'll have to figure it out yourself.**
